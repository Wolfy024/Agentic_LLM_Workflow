FEATURES

Everything you need to
build with AI

A comprehensive toolkit for autonomous coding agents, from file manipulation to git operations and beyond.

Autonomous Agent Loop

Self-directed reasoning with tool use, retry logic, and streaming responses. The agent plans, executes, and adapts in real-time.

Rich REPL Interface

Interactive terminal with slash commands — /model, /watch, /yolo and more.

12+ Tool Integrations

File system, git, GitHub API, web search, image generation, URL fetching, and subprocess execution — all at the agent's fingertips.

Permission Profiles

Three safety levels — strict, dev, ci — control what the agent can do.

Watch Mode

Auto-reload on file changes with auto or batch strategies. Never miss a change.

Full Git Integration

Branches, commits, diffs, blame, remotes, tags, and GitHub API — the agent can manage your entire git workflow.

ARCHITECTURE

DEPLOYMENT

Ship as a single binary

Distribute your agent as a standalone executable — no Python installation required. Choose your build tool.

PyInstaller

Most popular, best dependency detection
$ pip install pyinstaller
$ pyinstaller --onefile main.py
  • Single file output
  • Auto-detects dependencies
  • Cross-platform builds

Nuitka

Compiles to native C — 2-4x faster
$ pip install nuitka
$ nuitka --standalone --onefile main.py
  • Native compilation
  • 2-4x performance boost
  • Smaller output size

PyOxidizer

Embeds Python in Rust — zero overhead
$ pip install pyoxidizer
$ pyoxidizer init
$ pyoxidizer build release
  • Embeds CPython in binary
  • True single binary
  • Resource bundling built-in
